Instant Todo List API

GET /
- Return this page

GET /recipes
List all recipes
- Return an array of all the recipes width
  - id
  - name
  - time: time in minute or null
  - description: first 3 lines

POST /recipe
Create a Recipe
- Body
  - name
  - time: [optional] The time in minutes to execute the recipe
  - tags: an array of tags
  - ingredients: list of ingredients
    - name
    - quantity: a string to express how much of the ingredient is needed
  - description
- Return the created Recipe { id: "", name: "", ... }

GET /recipe/[id]
Get a Recipe by it's ID
- Return the recipe

PUT /recipe/[id]
Update a recipe
- Body: same as create execpt all properties are optional
- Return the updated recipe

DELETE /recipe/[id]
Delete a recipe
- Return an empty response

GET /tags
List all tags
- Return an list of all tags

GET /tag/[tag]
List all recipe for a tag
- Return an list of all recipe containing the tag
